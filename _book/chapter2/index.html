2. 数据类型

2. 数据类型

基本类型：

  • 空值 （null）
  • 布尔值（boolean）
  • 数字 （number）
  • 字符串（string）
  • 未定义（undefined）
  • symbol（symbol —— ES6新增）

引用类型：

Object:

  • Object 本质上是由一组无序的名值对组成。
  • 引用类型除 Object 外，还包括 Function 、Array、RegExp、Date 等等。
